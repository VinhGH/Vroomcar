# Frontend - Vroomcar

Dự án frontend sử dụng React.js + Vite.

## Cấu trúc thư mục
- `src/assets/`: Hình ảnh, icons, font.
- `src/components/ui/`: Base UI components (Button, Input, etc.).
- `src/components/common/`: Layout components (Header, Footer, etc.).
- `src/pages/`: Các trang của ứng dụng.
- `src/hooks/`: Custom React hooks.
- `src/contexts/`: Global state management.
- `src/services/`: API configuration and calls.
- `src/lib/`: Third-party library configurations and utility functions.
- `src/App.jsx`: Root component.
- `src/main.jsx`: Entry point.
